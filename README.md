# 📊 Financial Health Dashboard - Power BI

An interactive **Financial Health Dashboard** built using **Microsoft Power BI** to analyze an organization's financial performance. The dashboard provides insights into revenue, expenses, profitability, assets, liabilities, cash flow, and budget performance through dynamic visualizations and KPIs.

---

# 📌 Project Overview

The Financial Health Dashboard helps businesses monitor key financial indicators and make data-driven decisions. It consolidates data from multiple financial datasets into a single interactive dashboard, allowing users to evaluate profitability, cash flow, financial position, and budget utilization.

# 📈 Dashboard Insights

- Revenue reached **1 Million** during the reporting period.
- Net Profit totaled **504K**, indicating strong profitability.
- COGS accounted for **454K** of total revenue.
- Operating Expenses remained at **178K**.
- The dashboard analyzes **Operating, Investing, Financing, and Net Cash Flow**.
- Budgeted Revenue and Budgeted Expenses are compared with actual financial performance.
- Users can filter financial data using the interactive **Date Slicer**.
